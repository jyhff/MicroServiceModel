# 管理模块功能文档

## 1. 模块概述

| 项目 | 内容 |
|------|------|
| 模块名称 | AdminService（管理服务） |
| 功能定位 | 后台管理、用户管理、视频审核、数据统计、系统配置 |
| 创建时间 | 2026-05-12 |

---

## 2. 功能列表

### 2.1 管理员登录

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 管理员登录 |
| 功能描述 | 管理员登录后台系统 |
| 输入 | 管理员账号、密码 |
| 输出 | 管理员Token |
| 业务规则 | 仅超级管理员（用户需求） |
| 前置条件 | 账号为超级管理员 |
| 后置条件 | 登录成功、获得管理权限 |

---

### 2.2 数据统计仪表盘

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 数据统计仪表盘 |
| 功能描述 | 查看平台运营数据统计 |
| 输入 | 无 |
| 输出 | DAU、MAU、新增用户、留存率、播放量、上传量 |
| 业务规则 | 实时统计、历史数据查询 |
| 前置条件 | 管理员已登录 |
| 后置条件 | 返回统计数据 |

---

### 2.3 用户管理

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 用户管理 |
| 功能描述 | 查看用户列表、封禁用户、解封用户 |
| 输入 | 用户ID、操作类型 |
| 输出 | 操作成功确认 |
| 业务规则 | 支持账号封禁、上传封禁、评论封禁、弹幕封禁；支持临时封禁和永久封禁 |
| 前置条件 | 管理员已登录 |
| 后置条件 | 用户状态变化 |

---

### 2.4 视频审核管理

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频审核管理 |
| 功能描述 | 管理员审核视频（人工审核） |
| 输入 | 视频ID、审核决定（通过/拒绝）、审核意见 |
| 输出 | 审核成功确认 |
| 业务规则 | 审核通过可发布、审核拒绝需说明原因 |
| 前置条件 | 视频待审核 |
| 后置条件 | 视频状态变化 |

---

### 2.5 评论审核管理

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 评论审核管理 |
| 功能描述 | 管理员审核评论（人工审核） |
| 输入 | 评论ID、审核决定 |
| 输出 | 审核成功确认 |
| 业务规则 | 评论先审后发、审核通过可显示 |
| 前置条件 | 评论待审核 |
| 后置条件 | 评论状态变化 |

---

### 2.6 分类管理

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 分类管理 |
| 功能描述 | 创建、编辑、删除分类 |
| 输入 | 分类ID、分类名称、图标 |
| 输出 | 操作成功确认 |
| 业务规则 | 主分区最多10个、可启用/禁用分类 |
| 前置条件 | 管理员已登录 |
| 后置条件 | 分类状态变化 |

---

### 2.7 系统配置管理

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 系统配置管理 |
| 功能描述 | 管理系统参数配置 |
| 输入 | 配置键、配置值 |
| 输出 | 配置更新成功确认 |
| 业务规则 | 可配置各类系统参数 |
| 前置条件 | 管理员已登录 |
| 后置条件 | 系统配置更新 |

---

### 2.8 封禁记录查询

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 封禁记录查询 |
| 功能描述 | 查询用户封禁历史 |
| 输入 | 用户ID |
| 输出 | 封禁记录列表（封禁类型、原因、时间） |
| 业务规则 | 记录所有封禁操作历史 |
| 前置条件 | 管理员已登录 |
| 后置条件 | 返回封禁记录 |

---

## 3. API接口

| 功能 | API接口 |
|------|---------|
| 管理员登录 | POST /api/v1/admin/login |
| 数据统计 | GET /api/v1/admin/dashboard/stats |
| 用户列表 | GET /api/v1/admin/users |
| 用户封禁 | POST /api/v1/admin/users/{id}/ban |
| 用户解封 | POST /api/v1/admin/users/{id}/unban |
| 视频审核列表 | GET /api/v1/admin/videos/pending |
| 视频审核 | POST /api/v1/admin/videos/{id}/approve |
| 视频拒绝 | POST /api/v1/admin/videos/{id}/reject |
| 评论审核列表 | GET /api/v1/admin/comments/pending |

---

## 4. 封禁类型

| 类型 | 说明 |
|------|------|
| 账号封禁 | 用户账号无法登录 |
| 上传封禁 | 用户无法上传视频 |
| 评论封禁 | 用户无法发表评论 |
| 弹幕封禁 | 用户无法发送弹幕 |

---

## 5. 业务规则汇总

| 规则 | 说明 | 来源 |
|------|------|------|
| 管理员角色 | 仅超级管理员 | 用户需求 |
| 封禁类型 | 账号、上传、评论、弹幕 | 系统设计 |
| 封禁时长 | 可临时或永久 | 系统设计 |
| 视频审核 | 人工审核 | 系统设计 |
| 评论审核 | 人工审核 | 用户需求 |
| 分类数量 | 10个主分区 | 用户需求 |
