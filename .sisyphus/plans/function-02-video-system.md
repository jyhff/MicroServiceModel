# 视频模块功能文档

## 1. 模块概述

| 项目 | 内容 |
|------|------|
| 模块名称 | VideoService（视频服务） |
| 功能定位 | 视频上传、转码、存储、播放、管理全生命周期 |
| 创建时间 | 2026-05-12 |

---

## 2. 功能列表

### 2.1 视频上传（分片上传）

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频上传 |
| 功能描述 | 用户上传视频文件（分片上传） |
| 输入 | 文件名、文件大小、文件Hash、视频内容 |
| 输出 | 视频ID、上传状态、分片上传进度 |
| 业务规则 | 无时长限制、无大小限制（用户需求）、5MB分片、秒传检测、24小时会话过期 |
| 前置条件 | 用户为UP主、已登录 |
| 后置条件 | 视频上传成功、触发转码 |

---

### 2.2 视频秒传

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频秒传 |
| 功能描述 | 通过文件Hash检测，相同文件秒传 |
| 输入 | 文件Hash |
| 输出 | 秒传成功确认、视频ID |
| 业务规则 | 文件Hash匹配即秒传、无需重复上传 |
| 前置条件 | 文件Hash已存在于系统 |
| 后置条件 | 秒传成功、视频创建 |

---

### 2.3 视频转码

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频转码 |
| 功能描述 | CPU转码生成多种清晰度（360P、480P、720P、1080P、4K） |
| 输入 | 原始视频文件 |
| 输出 | 多清晰度转码文件、转码进度 |
| 业务规则 | CPU转码（用户需求）、所有清晰度免费、5个清晰度 |
| 前置条件 | 视频上传完成 |
| 后置条件 | 转码完成、可进入审核 |

---

### 2.4 视频审核

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频审核 |
| 功能描述 | 视频内容审核（人工审核） |
| 输入 | 视频ID |
| 输出 | 审核状态（通过/拒绝）、审核意见 |
| 业务规则 | 必须审核后才能发布、审核通过可发布 |
| 前置条件 | 视频转码完成 |
| 后置条件 | 审核完成、可发布或拒绝 |

---

### 2.5 视频发布

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频发布 |
| 功能描述 | 审核通过后发布视频 |
| 输入 | 视频ID |
| 输出 | 发布成功确认、发布时间 |
| 业务规则 | 审核通过才能发布、发布后可被用户观看 |
| 前置条件 | 视频审核通过 |
| 后置条件 | 视频发布成功、用户可观看 |

---

### 2.6 视频播放

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频播放 |
| 功能描述 | 用户播放视频（选择清晰度） |
| 输入 | 视频ID、清晰度选择 |
| 输出 | 视频播放URL、播放进度 |
| 业务规则 | 所有清晰度免费（用户需求）、记录播放进度、增加播放量 |
| 前置条件 | 视频已发布 |
| 后置条件 | 用户观看、播放量增加 |

---

### 2.7 视频管理（UP主）

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频管理 |
| 功能描述 | UP主管理自己的视频（编辑、删除、下架） |
| 输入 | 视频ID、操作类型 |
| 输出 | 操作成功确认 |
| 业务规则 | UP主可编辑标题、封面、分类；可下架、删除视频 |
| 前置条件 | 用户为UP主、视频属于该UP主 |
| 后置条件 | 视频状态变化 |

---

### 2.8 多分P视频管理

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 多分P视频管理 |
| 功能描述 | 上传和管理多分P视频（视频分章节） |
| 输入 | 多个视频文件、分P标题 |
| 输出 | 多分P视频ID、分P列表 |
| 业务规则 | 分P数量无限制、分P序号从1开始、已发布视频不能修改分P |
| 前置条件 | 用户为UP主 |
| 后置条件 | 多分P视频创建成功 |

---

### 2.9 视频封面设置

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频封面设置 |
| 功能描述 | 设置视频封面图（自动截取或自定义） |
| 输入 | 视频ID、封面URL或截取时间点 |
| 输出 | 封面URL |
| 业务规则 | 可自动截取、可自定义上传、封面尺寸建议16:9 |
| 前置条件 | 视频已上传 |
| 后置条件 | 封面设置成功 |

---

### 2.10 视频统计

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频统计 |
| 功能描述 | 视频播放量、点赞、收藏、投币、评论统计 |
| 输入 | 视频ID |
| 输出 | 播放量、点赞数、收藏数、投币数、评论数 |
| 业务规则 | 实时统计、热度计算：播放量*0.3 + 互动权重*0.7 |
| 前置条件 | 视频已发布 |
| 后置条件 | 返回统计数据 |

## 3. API接口映射

| 功能 | API接口 |
|------|---------|
| 初始化上传 | POST /api/v1/video/upload/init |
| 上传分片 | POST /api/v1/video/upload/chunk |
| 完成上传 | POST /api/v1/video/upload/complete |
| 秒传检测 | POST /api/v1/video/upload/instant |
| 获取视频信息 | GET /api/v1/video/{id} |
| 视频播放 | GET /api/v1/video/{id}/play |
| 更新视频信息 | PUT /api/v1/video/{id} |
| 删除视频 | DELETE /api/v1/video/{id} |
| 视频列表 | GET /api/v1/video/list |

| 360P | 360×640 | 800kbps | 免费 |
| 480P | 480×854 | 1.5Mbps | 免费 |
| 720P | 720×1280 | 3Mbps | 免费 |
| 1080P | 1080×1920 | 5Mbps | 免费 |

## 5. 视频状态流转

```
Uploaded（已上传）
   ↓
Transcoding（转码中）
   ↓
Transcoded（转码完成）
   ↓
Auditing（审核中）
   ↓
PendingPublish（待发布）
   ↓
Published（已发布）←→Unpublished（已下架）
   ↓
Deleted（已删除）
```
