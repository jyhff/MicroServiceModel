# 直播模块功能文档

## 1. 模块概述

| 项目 | 内容 |
|------|------|
| 模块名称 | LiveService（直播服务） |
| 功能定位 | 直播开播、推流、播放、观众管理、礼物打赏 |
| 创建时间 | 2026-05-12 |

---

## 2. 功能列表

### 2.1 开播

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 开播 |
| 功能描述 | 用户开始直播 |
| 输入 | 直播标题、分类ID、封面 |
| 输出 | 直播ID、推流密钥、RTMP推流地址、HLS播放地址 |
| 业务规则 | 所有用户可开播（用户需求）、生成推流密钥 |
| 前置条件 | 用户已登录 |
| 后置条件 | 直播创建成功、可推流 |

---

### 2.2 推流

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 推流 |
| 功能描述 | 主播使用推流软件推流到直播服务器 |
| 输入 | RTMP推流地址、推流密钥、视频流 |
| 输出 | 推流成功确认 |
| 业务规则 | RTMP协议、密钥验证、密钥24小时过期 |
| 前置条件 | 直播已开播 |
| 后置条件 | 推流成功、观众可观看 |

---

### 2.3 直播播放

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 直播播放 |
| 功能描述 | 观众观看直播 |
| 输入 | 直播ID |
| 输出 | HLS播放地址、直播信息 |
| 业务规则 | HLS延迟<10秒（用户需求） |
| 前置条件 | 直播正在直播中 |
| 后置条件 | 观众可观看直播 |

---

### 2.4 礼物打赏

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 礼物打赏 |
| 功能描述 | 观众向主播送礼物 |
| 输入 | 直播ID、礼物ID、礼物数量、赠言 |
| 输出 | 打赏成功确认、UP主获得金额 |
| 业务规则 | 礼物100%给UP主（用户需求）、扣减用户B币 |
| 前置条件 | 用户已登录、B币余额充足 |
| 后置条件 | 礼物发送成功、主播获得B币 |

---

### 2.5 观众进入/离开

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 观众管理 |
| 功能描述 | 观众进入和离开直播间 |
| 输入 | 直播ID、用户ID |
| 输出 | 观众数变化确认 |
| 业务规则 | 实时观众数统计、累计观众数 |
| 前置条件 | 用户已登录 |
| 后置条件 | 观众数更新 |

---

### 2.6 直播统计

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 直播统计 |
| 功能描述 | 直播观众数、礼物收入统计 |
| 输入 | 直播ID |
| 输出 | 当前观众数、累计观众数、峰值观众数、礼物总收入 |
| 业务规则 | 实时观众数从Redis同步 |
| 前置条件 | 直播存在 |
| 后置条件 | 返回统计数据 |

---

### 2.7 结束直播

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 结束直播 |
| 功能描述 | 主播结束直播 |
| 输入 | 直播ID |
| 输出 | 结束确认、直播统计数据 |
| 业务规则 | 结束后统计数据永久保存 |
| 前置条件 | 直播正在直播中 |
| 后置条件 | 直播结束 |

---

## 3. API接口

| 功能 | API接口 |
|------|---------|
| 开播 | POST /api/v1/live/start |
| 结束直播 | POST /api/v1/live/end |
| 直播列表 | GET /api/v1/live/list |
| 直播详情 | GET /api/v1/live/{id} |
| 送礼物 | POST /api/v1/live/gift |

---

## 4. 礼物分成规则

| 项目 | 说明 |
|------|------|
| UP主获得 | 100%（用户需求） |
| 平台获得 | 0% |

---

## 5. 业务规则汇总

| 规则 | 说明 | 来源 |
|------|------|------|
| 开播权限 | 所有用户可开播 | 用户需求 |
| 直播延迟 | HLS<10秒 | 用户需求 |
| 礼物分成 | 100%给UP主 | 用户需求 |
| 推流协议 | RTMP | 系统设计 |
| 播放协议 | HLS | 系统设计 |
| 密钥过期 | 24小时 | 系统设计 |
