# 搜索模块功能文档

## 1. 模块概述

| 项目 | 内容 |
|------|------|
| 模块名称 | SearchService（搜索服务） |
| 功能定位 | 视频搜索、搜索建议、搜索历史、索引管理 |
| 创建时间 | 2026-05-12 |

---

## 2. 功能列表

### 2.1 视频搜索

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 视频搜索 |
| 功能描述 | 搜索视频（标题、描述、UP主昵称） |
| 输入 | 关键词、分类、排序、分页 |
| 输出 | 搜索结果列表（含高亮、得分） |
| 业务规则 | Elasticsearch全文搜索、ik_max_word分词、实时索引同步 |
| 前置条件 | 关键词有效 |
| 后置条件 | 返回搜索结果 |

---

### 2.2 搜索排序

| 排序类型 | 说明 |
|---------|------|
| 相关性 | 综合排序（相关性70% + 热度30%） |
| 播放量 | 播放量降序 |
| 发布时间 | 最新发布降序 |
| 点赞数 | 点赞数降序 |
| 收藏数 | 收藏数降序 |
| 时长 | 时长排序 |

---

### 2.3 搜索建议

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 搜索建议 |
| 功能描述 | 输入关键词时提供搜索建议 |
| 输入 | 输入关键词片段 |
| 输出 | 建议关键词列表 |
| 业务规则 | 用户历史优先、热门关键词补充、最多10条 |
| 前置条件 | 无 |
| 后置条件 | 返回建议列表 |

---

### 2.4 搜索历史

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 搜索历史 |
| 功能描述 | 记录用户搜索历史 |
| 输入 | 用户ID |
| 输出 | 用户搜索历史列表 |
| 业务规则 | 记录关键词、搜索次数、可清除历史 |
| 前置条件 | 用户已登录 |
| 后置条件 | 返回搜索历史 |

---

### 2.5 热门搜索

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 热门搜索 |
| 功能描述 | 获取当前热门搜索关键词 |
| 输入 | 无 |
| 输出 | 热门关键词列表 |
| 业务规则 | 按搜索次数排序、最多10条 |
| 前置条件 | 无 |
| 后置条件 | 返回热门关键词 |

---

## 3. API接口

| 功能 | API接口 |
|------|---------|
| 视频搜索 | GET /api/v1/search |
| 搜索建议 | GET /api/v1/search/suggest |
| 搜索历史 | GET /api/v1/search/history |
| 热门搜索 | GET /api/v1/search/hot |

---

## 4. Elasticsearch索引字段

| 字段 | 类型 | 说明 |
|------|------|------|
| videoId | keyword | 视频ID |
| title | text (ik_max_word) | 视频标题 |
| description | text (ik_max_word) | 视频描述 |
| nickname | text (ik_max_word) | UP主昵称 |
| categoryId | keyword | 分类ID |
| viewCount | long | 播放量 |
| publishTime | date | 发布时间 |

---

| 规则 | 说明 | 来源 |
|------|------|------|
