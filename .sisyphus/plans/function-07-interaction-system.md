# 互动模块功能文档

## 1. 模块概述

| 项目 | 内容 |
|------|------|
| 模块名称 | InteractionService（互动服务） |
| 功能定位 | 点赞、收藏、投币、评论、分享、观看历史 |
| 创建时间 | 2026-05-12 |

---

## 2. 功能列表

### 2.1 点赞

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 点赞 |
| 功能描述 | 用户点赞视频或评论 |
| 输入 | 目标ID、目标类型（视频/评论） |
| 输出 | 点赞成功确认、点赞数 |
| 业务规则 | 不能重复点赞、可取消点赞 |
| 前置条件 | 用户已登录 |
| 后置条件 | 点赞数增加 |

---

### 2.2 取消点赞

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 取消点赞 |
| 功能描述 | 用户取消点赞 |
| 输入 | 目标ID、目标类型 |
| 输出 | 取消成功确认、点赞数 |
| 业务规则 | 已点赞才能取消 |
| 前置条件 | 用户已点赞 |
| 后置条件 | 点赞数减少 |

---

### 2.3 收藏

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 收藏 |
| 功能描述 | 用户收藏视频 |
| 输入 | 视频ID、收藏夹ID（可选）、收藏备注（可选） |
| 输出 | 收藏成功确认、收藏数 |
| 业务规则 | 可收藏到指定收藏夹、可添加备注 |
| 前置条件 | 用户已登录 |
| 后置条件 | 收藏数增加 |

---

### 2.4 投币

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 投币 |
| 功能描述 | 用户投币给视频（UP主） |
| 输入 | 视频ID、投币数量（1或2） |
| 输出 | 投币成功确认、UP主投币数、用户余额 |
| 业务规则 | 投币数量无限制（用户需求）、单次最多2个、100%给UP主 |
| 前置条件 | 用户已登录、B币余额充足 |
| 后置条件 | UP主B币增加、用户B币减少 |

---

### 2.5 评论（先审后发）

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 评论 |
| 功能描述 | 用户发表评论（先审后发） |
| 输入 | 视频ID、评论内容、父评论ID（回复） |
| 输出 | 评论ID、审核状态（待审核） |
| 业务规则 | 评论先审后发（人工审核）（用户需求）、内容≤1000字符、最多3层回复 |
| 前置条件 | 用户已登录 |
| 后置条件 | 评论提交、等待审核 |

---

### 2.6 评论审核

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 评论审核 |
| 功能描述 | 管理员审核评论 |
| 输入 | 评论ID、审核结果（通过/拒绝）、审核意见 |
| 输出 | 审核确认 |
| 业务规则 | 通过后评论发布、拒绝后评论不显示 |
| 前置条件 | 评论待审核 |
| 后置条件 | 评论状态变化 |

---

### 2.7 分享

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 分享 |
| 功能描述 | 用户分享视频到各平台 |
| 输入 | 视频ID、分享平台 |
| 输出 | 分享成功确认、分享链接 |
| 业务规则 | 记录分享次数、支持多平台 |
| 前置条件 | 用户已登录 |
| 后置条件 | 分享数增加 |

---

### 2.8 观看历史

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 观看历史 |
| 功能描述 | 记录用户观看进度 |
| 输入 | 视频ID、观看进度、观看时长 |
| 输出 | 历史记录确认 |
| 业务规则 | 实时记录进度、可清除历史 |
| 前置条件 | 用户已登录 |
| 后置条件 | 观看历史更新 |

---

## 4. API接口

| 功能 | API接口 |
|------|---------|
| 点赞 | POST /api/v1/interaction/like |
| 取消点赞 | POST /api/v1/interaction/unlike |
| 收藏 | POST /api/v1/interaction/favorite |
| 取消收藏 | DELETE /api/v1/interaction/favorite |
| 投币 | POST /api/v1/interaction/coin |
| 评论 | POST /api/v1/interaction/comment |
| 评论列表 | GET /api/v1/interaction/comments |
| 分享 | POST /api/v1/interaction/share |
| 观看历史 | GET /api/v1/interaction/history |

---

## 5. 业务规则汇总

| 规则 | 说明 | 来源 |
|------|------|------|
| 投币上限 | 无限制 | 用户需求 |
| 评论审核 | 先审后发（人工审核） | 用户需求 |
| 投币分成 | 100%给UP主 | 用户需求 |
| 评论层级 | 最多3层 | 系统设计 |
| 评论长度 | ≤1000字符 | 系统设计 |
