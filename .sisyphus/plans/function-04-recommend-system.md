# 推荐模块功能文档

## 1. 模块概述

| 项目 | 内容 |
|------|------|
| 模块名称 | RecommendService（推荐服务） |
| 功能定位 | 首页推荐、协同过滤、热门推荐、个性化推荐 |
| 创建时间 | 2026-05-12 |

---

## 2. 功能列表

### 2.1 首页推荐

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 首页推荐 |
| 功能描述 | 为用户生成首页视频推荐列表 |
| 输入 | 用户ID、分页参数 |
| 输出 | 推荐视频列表（含推荐理由、推荐得分） |
| 业务规则 | 协同过滤+热门（用户需求）、已观看视频过滤 |
| 前置条件 | 用户已登录 |
| 后置条件 | 返回推荐列表 |

---

### 2.2 协同过滤推荐

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 协同过滤推荐 |
| 功能描述 | 基于用户行为相似性推荐视频 |
| 输入 | 用户ID |
| 输出 | 相似用户喜欢的视频列表 |
| 业务规则 | Jaccard/余弦相似度、相似用户>50个、共同视频>5个 |
| 前置条件 | 用户有行为历史 |
| 后置条件 | 返回协同过滤推荐 |

---

### 2.3 热门推荐

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 热门推荐 |
| 功能描述 | 基于热度排序推荐视频 |
| 输入 | 分类ID（可选） |
| 输出 | 热门视频列表 |
| 业务规则 | 热度=播放量*0.3 + 互动权重*0.7 |
| 前置条件 | 无 |
| 后置条件 | 返回热门视频列表 |

---

### 2.4 用户行为记录

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 用户行为记录 |
| 功能描述 | 记录用户观看、点赞、收藏、分享、投币行为 |
| 输入 | 用户ID、视频ID、行为类型 |
| 输出 | 行为记录确认 |
| 业务规则 | 不同行为有不同权重、实时同步Elasticsearch |
| 前置条件 | 用户已登录 |
| 后置条件 | 行为记录、用户画像更新 |

---

### 2.5 用户画像更新

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 用户画像更新 |
| 功能描述 | 根据用户行为更新用户兴趣画像 |
| 输入 | 用户行为数据 |
| 输出 | 用户画像（分类偏好、UP主偏好） |
| 业务规则 | 分类偏好得分累加、每日衰减因子0.9 |
| 前置条件 | 用户有行为数据 |
| 后置条件 | 用户画像更新 |

---

## 3. 行为权重表

| 行为 | 权重值 | 说明 |
|------|--------|------|
| 观看<30% | 1.0 | 低权重 |
| 观看30-80% | 3.0 | 中权重 |
| 观看>80% | 5.0 | 高权重 |
| 点赞 | 3.0 | 中权重 |
| 收藏 | 5.0 | 高权重 |
| 分享 | 5.0 | 高权重 |
| 投币 | 10.0+5/币 | 极高权重 |
| 评论 | 4.0 | 中权重 |

---

## 4. API接口

| 功能 | API接口 |
|------|---------|
| 首页推荐 | GET /api/v1/recommendation/feed |
| 热门推荐 | GET /api/v1/recommendation/hot |
| 用户行为记录 | POST /api/v1/recommendation/behavior |

## 5. 业务规则汇总

| 规则 | 说明 | 来源 |
|------|------|------|
| 推荐算法 | 协同过滤+热门 | 用户需求 |

| 协同权重 | 40% | 系统设计 |
