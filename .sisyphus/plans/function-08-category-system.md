# 分类管理模块功能文档

## 1. 模块概述

| 项目 | 内容 |
|------|------|
| 模块名称 | CategoryService（分类服务） |
| 功能定位 | 分类管理、内容审核（AI自动审核） |
| 创建时间 | 2026-05-12 |

---

## 2. 功能列表

### 2.1 分类列表

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 分类列表 |
| 功能描述 | 获取视频分类列表 |
| 输入 | 无 |
| 输出 | 分类列表（主分区+子分区） |
| 业务规则 | 10个主分区（用户需求）、主分区下有子分区 |
| 前置条件 | 无 |
| 后置条件 | 返回分类列表 |

---

### 2.2 分类详情

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 分类详情 |
| 功能描述 | 获取分类详细信息 |
| 输入 | 分类ID |
| 输出 | 分类名称、图标、子分类列表 |
| 业务规则 | 主分区有子分区、子分区属于主分区 |
| 前置条件 | 分类ID有效 |
| 后置条件 | 返回分类详情 |

---

### 2.3 分类管理（管理员）

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 分类管理 |
| 功能描述 | 管理员创建、编辑、禁用分类 |
| 输入 | 分类名称、图标、排序 |
| 输出 | 操作成功确认 |
| 业务规则 | 主分区最多10个、可启用/禁用分类 |
| 前置条件 | 用户为管理员 |
| 后置条件 | 分类状态变化 |

---

### 2.4 AI内容审核

| 功能项 | 说明 |
|--------|------|
| 功能名称 | AI内容审核 |
| 功能描述 | AI自动审核视频内容 |
| 输入 | 视频ID、视频内容 |
| 输出 | AI审核结果（通过/需人工审核） |
| 业务规则 | AI自动审核通过（用户需求）、标记可疑内容需人工审核 |
| 前置条件 | 视频已上传 |
| 后置条件 | 审核状态变化 |

---

### 2.5 人工内容审核

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 人工内容审核 |
| 功能描述 | 管理员人工审核AI标记的内容 |
| 输入 | 视频ID、审核结果（通过/拒绝）、审核意见 |
| 输出 | 审核确认 |
| 业务规则 | AI标记的内容需人工审核 |
| 前置条件 | 视频需人工审核 |
| 后置条件 | 审核完成 |

---

## 4. API接口

| 功能 | API接口 |
|------|---------|
| 分类列表 | GET /api/v1/categories |
| 分类详情 | GET /api/v1/categories/{id} |
| 视频审核 | POST /api/v1/content/audit/{videoId} |

---

## 5. 业务规则汇总

| 规则 | 说明 | 来源 |
|------|------|------|
| 主分区数量 | 10个 | 用户需求 |
| 视频审核 | AI自动审核通过 | 用户需求 |
| 分类层级 | 主分区→子分区 | 系统设计 |
| 分类启用 | 支持启用/禁用 | 系统设计 |
