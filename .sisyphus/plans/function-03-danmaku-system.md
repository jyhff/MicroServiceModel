# 弹幕模块功能文档

## 1. 模块概述

| 项目 | 内容 |
|------|------|
| 模块名称 | DanmakuService（弹幕服务） |
| 功能定位 | 弹幕发送、显示、过滤、存储、实时推送 |
| 创建时间 | 2026-05-12 |

---

## 2. 功能列表

### 2.1 WebSocket连接

| 功能项 | 说明 |
|--------|------|
| 功能名称 | WebSocket连接 |
| 功能描述 | 用户连接弹幕房间，实时接收和发送弹幕 |
| 输入 | 视频ID、JWT Token |
| 输出 | 连接成功确认、房间观众数、最近100条弹幕 |
| 业务规则 | JWT认证、每个视频独立房间 |
| 前置条件 | 用户已登录 |
| 后置条件 | 连接成功、可发送和接收弹幕 |

---

### 2.2 发送弹幕

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 发送弹幕 |
| 功能描述 | 用户发送弹幕到视频 |
| 输入 | 视频ID、弹幕内容、时间点、类型、颜色、字号 |
| 输出 | 弹幕ID、弹幕显示确认 |
| 业务规则 | 内容≤100字符（高级≤150）、LV2+可用高级弹幕 |
| 前置条件 | 用户已登录、已连接房间 |
| 后置条件 | 弹幕广播到所有房间用户 |

---

### 2.3 弹幕类型

| 类型 | 编号 | 说明 |
|------|------|------|
| 滚动弹幕 | 0 | 从右到左滚动，5秒显示 |
| 顶部弹幕 | 1 | 固定在顶部，3秒显示 |
| 底部弹幕 | 2 | 固定在底部，3秒显示 |
| 逆向弹幕 | 3 | 高级弹幕：从左到右 |
| 精准弹幕 | 4 | 高级弹幕：精确定位 |
| 自定义 | 5 | 高级弹幕：自定义样式 |

---

### 2.4 弹幕过滤

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 弹幕过滤 |
| 功能描述 | 用户设置弹幕过滤规则 |
| 输入 | 过滤规则（时间范围、类型、颜色、用户屏蔽） |
| 输出 | 过滤后弹幕列表 |
| 业务规则 | 支持屏蔽用户、屏蔽关键词、屏蔽高级弹幕 |
| 前置条件 | 用户已登录 |
| 后置条件 | 弹幕按过滤规则显示 |

---

### 2.5 弹幕查询

| 功能项 | 说明 |
|--------|------|
| 功能名称 | 弹幕查询 |
| 功能描述 | 查询指定时间段的弹幕 |
| 输入 | 视频ID、开始时间、结束时间 |
| 输出 | 弹幕列表 |
| 业务规则 | PostgreSQL分区表存储、按分段查询（10秒分段） |
| 前置条件 | 视频ID有效 |
| 后置条件 | 返回弹幕列表 |

---

## 3. API/WebSocket接口

| 功能 | 接口 |
|------|------|
| WebSocket连接 | wss://api.bilibili.com/ws/danmaku?videoId={id} |
| 发送弹幕 | WebSocket消息：{type: "send"} |
| 弹幕查询 | GET /api/v1/danmaku/list?videoId={id} |

---

## 4. 业务规则汇总

| 规则 | 说明 | 来源 |
|------|------|------|
| 弹幕并发 | 无限制 | 用户需求 |
| 存储方式 | PostgreSQL分区表 | 用户需求 |
| 高级弹幕 | LV2+可用 | 用户需求 |
| 弹幕长度 | 普通≤100，高级≤150 | 系统设计 |
| Redis缓存 | 最近100条实时缓冲 | 系统设计 |
| 分段策略 | 每10秒一个分段 | 系统设计 |
