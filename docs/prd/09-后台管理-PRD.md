# 后台管理系统PRD

## 1. 功能模块

### 1.1 内容管理
- 视频审核列表
- 视频上下架
- 分区管理
- 标签管理

### 1.2 用户管理
- 用户列表
- 用户封禁
- 权限管理
- 用户等级管理

### 1.3 数据统计
- 播放统计
- 用户增长
- 收入统计
- 内容分析

## 2. 数据统计指标

| 指标 | 说明 |
|------|------|
| 日活(DAU) | 每日活跃用户数 |
| 月活(MAU) | 每月活跃用户数 |
| 日均播放 | 每日总播放量 |
| 平均观看时长 | 用户平均观看时长 |
| 完播率 | 视频看完比例 |
| 新用户留存 | 次日/7日/30日留存 |
| 投稿数 | 每日新视频数 |
| 审核通过率 | 视频审核通过比例 |

## 3. 技术栈

- 前端: Vue 3 + Ant Design Vue
- 后端: ABP Framework v10.0.2 + .NET 10.0
- 报表: ECharts
